# Dice/Игра в кости без 

* при старте у игрока всегда 500 очков
* игрок ставит от 1 до 12
* если угадал то выигрыш удваивается
* если проиграл то теряет сколько поставил
* игра бесконечная, без возможности выхода
* счетчик раундов

### Версия Ruby:
``` ruby 2.4.1 ```

### Для запуска:
``` game.rb ```
